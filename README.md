# Bank Queue Simulation (OS/CAO Concept) 🏦

A Python desktop application that actively demonstrates the **First-Come, First-Served (FCFS)** CPU Scheduling algorithm using a Bank Queue analogy.

## Theory & Analogy
- **Customer** = Process
- **Cashier** = CPU
- **Queue** = Ready Queue

## Features
- **Native GUI:** Built with Python's built-in `Tkinter` module.
- **Dynamic Table:** Calculates Waiting Time, Turnaround Time, and CPU Utilization instantly.
- **Visuals:** Generates informative bar charts using `matplotlib`.

## How to Run
Make sure you have Python installed, then run the following in your terminal:

```bash
# Install the required libraries for graphing and tables
pip install -r requirements.txt

# Run the application
python main.py
```
